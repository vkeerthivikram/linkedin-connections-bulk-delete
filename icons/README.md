# Icon Files

This directory should contain the following icon files for the Chrome extension:

- `icon16.png` - 16x16 pixels (for browser toolbar)
- `icon48.png` - 48x48 pixels (for extension management page)
- `icon128.png` - 128x128 pixels (for Chrome Web Store)

## Icon Design Guidelines

- Use LinkedIn's brand color (#0a66c2) as the primary color
- Include a trash/delete icon symbol
- Keep the design simple and professional
- Use transparent background for PNG files

## Placeholder Icons

If you don't have custom icons, you can use the following approach:

1. Create simple icons using an image editor
2. Use online icon generators
3. Use the default Chrome extension icon (will be generated automatically)

## Recommended Icon Design

A simple design with:
- Blue background (#0a66c2)
- White trash can icon in the center
- Clean, modern look

You can create these icons using tools like:
- Figma
- Canva
- Adobe Photoshop
- GIMP
- Online icon generators
